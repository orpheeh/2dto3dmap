Tutorial

1. Créer un document

Cliquez sur le bouton Ajouter dans la rubrique Batiment en bas à gauche de l'écran

2. Dessiner le batiment

Sélectionnez le batiment dans la rubrique Batiment

Dessinez un rectangle dans la zone de dessin à gauche Vous devrez voir un batiment en 3D dans la zone de droite

3.Dessiner l'interieur du batiment

Choisissez le mode de dessin Interieur dans la rubrique Que dessiner

Dessinez un mur à l'interieur du carré précédent

4. Ajouter un étage

Cliquez 3 fois sur le bouton Ajouter dans la rubrique Ascnesseur pour ajouter 3 étages au batiment

Cliquez sur le bouton 1 pour pouvoir dessiner le plan du premier étage

Faite le plan de cette étage

5. Supprimer un étage

Cliquez sur le bouton Retirer une fois pour supprimer le dernier étage

6.supprimer le dernier mur intérieur ou le dernier point du contour

Sélectionnez Contour dans la rubrique Que dessiner

Appuiyez 2 fois sur le bouton A du clavier

Bouchez la souris

Fini !

Maintenant vous savez comment utiliser 2D to 3D

Commandes de base

Rubrique Batiment

La rubrique batiment permet de gerer les batiments. dans cette rubrique on peut:

  • Ajouter un batiment Ajouter
  • Supprimer un batiment Retirer
  • Sélectionner le batiment sur lequel effectuer les modifications

Rubrique Que dessiner

La rubrique Que dessiner permet de choisir si on veut dessiner les contours du batiment ou le plan interieur d'un de ses étages

Rubrique Ascensseur

La rubrique Ascensseur permet de gérer les différent étage du batiment sélectionné

  • Le bouton Ajouter permet d'ajouter un étage au batiment
  • Le bouton Retirer permet de supprimer le dernier étage du batiment
  • Les numero 0 1 ... 5 représentes les 5 étage possible que peut avoir le batiment
  • Les étages que le batiment poccède sont représenté par un nombre noir sur un fond gris
  • L'étage courant est représenté par un nombre noir sur un fond bleu

Le bouton A du clavier

  • Le bouton A permet de supprimer le dernier point du contour du batiment quand Que dessiner est à Contour
  • Le bouton A permet de supprimer le dernier mur de l'étage sélectionné du batiment quand Que dessiner est à Intérieur
